Routes for you

Method Route Description Example
POST /movies Takes title of a movie as argument (in JSON file), and saves it to database. Example request body: {'title':'The Shining'}
GET /movies Returns all movies in database. -
POST /comment Takes movie id and comment as arguments and add comment to database. Example request body: {'movie':1,'comment':'Cool!'}
GET /comment Returns all coments from database. -
GET /top Returns ranking of movies from specified period of time. Example request body: /top/?date_start=2011-10-21&date_end=2020-10-21
